# BMI Calculator

A flutter based Body mass index calculator and also a app to share the report and gives the required information to maintain BMI in good levels



```bash
BMI Categories:

Low : bmi < 18.5
Normal : 18.4 < bmi < 25.1
High : bmi > 25
```
